# Rich Text Editor - User Guide for Church Staff

## Overview
Your Church Digital Engagement Platform now includes a professional rich text editor similar to Microsoft Word. This allows you to create beautifully formatted posts with headings, colors, images, and more!

## Accessing the Editor

### Creating New Posts
1. Log in to the admin portal
2. Navigate to **Content Management**
3. Click **"Create New Post"**
4. You'll see the rich text editor below the title field

### Editing Existing Posts
1. Go to **Content Management**
2. Find the post you want to edit
3. Click the **Edit** button (pencil icon)
4. The editor will load with your existing content

### Basic Text Formatting

**Bold Text**
- Click the **B** button or press `Ctrl+B` (Windows) / `Cmd+B` (Mac)
- Use for emphasis and important points

**Italic Text**
- Click the **I** button or press `Ctrl+I` / `Cmd+I`
- Use for quotes, scripture references, or subtle emphasis

**Underline**
- Click the **U** button or press `Ctrl+U` / `Cmd+U`
- Use sparingly for important information

**Strikethrough**
- Click the **S** button
- Great for showing price changes or canceled events

### Headings

Use the **Heading** dropdown to create different heading levels:

- **Heading 1**: Main title (largest)
- **Heading 2**: Major sections
- **Heading 3**: Subsections
- **Heading 4**: Minor sections
- **Normal**: Regular text

**Example Structure:**
```
Heading 1: Sunday Service Details
  Heading 2: Worship Times
    Normal text: Service begins at 10:00 AM
  Heading 2: Special Guest Speaker
    Normal text: Join us for...
```

## Common Mistakes to Avoid

‚ùå **Using too many colors**
- Stick to 2-3 colors maximum
- Use church brand colors when possible

‚ùå **ALL CAPS TEXT**
- Avoid using all capital letters
- Use bold or colors for emphasis instead

‚ùå **Too many font sizes**
- Use the preset heading levels
- Avoid manually changing sizes

‚ùå **Not using headings**
- Headings help organize content
- Makes posts easier to read

‚ùå **Pasting from Word with weird formatting**
- The editor cleans this automatically
- But review and adjust if needed

## Saving Your Work

### As Draft
1. Select **"Save as Draft"** from the status dropdown
2. Click **"Create Post"** or **"Update Post"**
3. Draft is saved but not visible to members

### Publishing
1. Select **"Publish Immediately"** from the status dropdown
2. Click **"Create Post"** or **"Update Post"**
3. Post goes live instantly!

Or for existing drafts:
1. Edit the post
2. Click **"Publish Now"** button
3. Confirm publication

## Tips for Great Posts

### 1. Start with a Compelling Title
Make it clear and interesting: "Easter Service Times" is better than "Service"

### 2. Use Images
Posts with images get more attention and engagement

### 3. Keep Paragraphs Short
3-4 sentences per paragraph makes reading easier

### 4. Use Formatting Sparingly
Too much bold/color becomes distracting

### 5. Include a Call to Action
"Register today!" or "Join us this Sunday!"

### 6. Proofread Before Publishing
Use spell check and read through once

### 7. Preview on Mobile
Many members read on phones - keep that in mind

## Troubleshooting

### "My formatting isn't showing up"
- Make sure you're selecting text before applying formatting
- Try refreshing the page and editing again

### "Images won't upload"
- Check file size (must be under 5MB)
- Try a different image format (JPG or PNG)
- Check your internet connection

### "I accidentally deleted everything"
- Click the Undo button (‚Ü∂) multiple times
- Or close without saving and reopen the editor

### "The editor looks strange"
- Try refreshing your browser
- Clear browser cache
- Try a different browser (Chrome recommended)

### "Content looks different after publishing"
- This shouldn't happen - contact your admin
- The display should match exactly what you created

## Getting Help

If you need assistance:
1. Check this guide first
2. Ask your admin or tech team
3. Take a screenshot of any errors
4. Note what you were trying to do when the issue occurred
